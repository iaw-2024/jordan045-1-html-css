La bebida oficial de los viernes

La cerveza, una bebida ancestral con miles de años de historia, se elabora con cereales, agua, lúpulo y levadura. Con una gran variedad de tipos y sabores, la cerveza puede aportar beneficios para la salud como fuente de vitaminas y minerales, antioxidantes, y por su efecto positivo en la salud cardiovascular y ósea. Siempre con moderación, la cerveza puede ser una bebida placentera y saludable.

Estilos de cerveza

  • Lager

    • Color: Amarillo claro
    • Sabor: Ligero, refrescante
    • Amargor (IBU): Bajo (10-20)
    • Alcohol (ABV): 4-6%
    • Ejemplos: Pilsner, Helles

  • Pale Ale

    • Color: Dorado
    • Sabor: Afrutado, cítrico
    • Amargor (IBU): Moderado (20-40)
    • Alcohol (ABV): 4-6%
    • Ejemplos: American Pale Ale, India Pale Ale

  • IPA

    • Color: Dorado a ámbar
    • Sabor: Afrutado, cítrico, resinoso
    • Amargor (IBU): Alto (40-100+)
    • Alcohol (ABV): 5-7%
    • Ejemplos: New England IPA, Double IPA

  • Stout

    • Color: Negro
    • Sabor: Tostado, chocolate, café
    • Amargor (IBU): Moderado (30-60)
    • Alcohol (ABV): 4-8%
    • Ejemplos: Dry Stout, Imperial Stout

  • Porter

    • Color: Marrón oscuro
    • Sabor: Tostado, chocolate, caramelo
    • Amargor (IBU): Moderado (20-40)
    • Alcohol (ABV): 5-7%
    • Ejemplos: Porter, Baltic Porter

  • Trigo

    • Color: Amarillo pajizo
    • Sabor: Afrutado, especiado, ácido
    • Amargor (IBU): Bajo (10-20)
    • Alcohol (ABV): 4-6%
    • Ejemplos: Hefeweizen, Belgian Witbier

  • Lambic

    • Color: Dorado a ámbar
    • Sabor: Ácido, agridulce, especiado
    • Amargor (IBU): Bajo (10-20)
    • Alcohol (ABV): 5-8%
    • Ejemplos: Kriek, Faro

  • Saison

    • Color: Dorado a ámbar
    • Sabor: Especiado, terroso, afrutado
    • Amargor (IBU): Moderado (20-40)
    • Alcohol (ABV): 5-7%
    • Ejemplos: Saison, Mosaic
